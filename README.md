**CodeArena** is a full-stack web application designed to gamify coding practice. It provides a unified ecosystem for developers to compete in 1v1 coding battles, communicate in real-time, and receive intelligent AI assistance‚Äîeliminating the need to switch between coding platforms and communication tools.

## üöÄ Key Features

* **‚öîÔ∏è 1v1 Coding Battles:** Create private rooms, generate unique invite codes, and race friends to solve algorithmic challenges in real-time.
* **‚ö° Real-Time Communication:** Integrated low-latency chat using **Spring WebSockets (STOMP)** allows users to communicate instantly within battle rooms.
* **ü§ñ AI-Powered Assistance:**
    * **Smart Hints:** Get nudges in the right direction without revealing the full solution.
    * **Code Review:** Receive JSON-formatted optimization tips and readability scores (1-10) generated by LLMs via **Spring AI**.
* **üèÉ Secure Code Execution:** Supports compilation and execution for **Java, Python, and C++** with automated test-case validation.
* **üîí Robust Security:** Stateless authentication using **JWT (JSON Web Tokens)**, optimistic locking for room consistency, and OTP-based email verification.
* **‚ö° High Performance:** Utilizes **Redis** for caching active room sessions and managing battle states efficiently.

## üèóÔ∏è Architecture Overview

The application follows a **Modular Monolithic** architecture optimized for real-time interaction:

1.  **Client Layer:** React Frontend handles UI and establishes WebSocket connections.
2.  **Controller Layer:** REST endpoints (`AuthController`, `RoomController`) manage traffic.
3.  **Service Layer:** Handles business logic (Code Compilation, AI Prompt Engineering).
4.  **Data Layer:** JPA Repositories for MySQL and `RedisTemplate` for session management.
